KimWeb 1.0
===============

> 运行环境要求PHP7.3+。

## 主要新特性

* 基于KimFramework
* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 原生多应用支持
* 适配Packagist高人气组件
* 分离能分离的一切，大道至简，用户可按需加载需要的组件及扩展
* 注重对IDE的友好，明确的参数提示，代码洁癖者的最佳选择

## 安装

~~~
composer create-project jinyicheng/kim-web project_name 1.0.*-dev
~~~

如果需要更新框架使用
~~~
composer update jinyicheng/kim-framework
~~~
